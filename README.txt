== Description ==

Customized WordPress login page with unique animation and effects.


== Installation ==

1. Upload `animated-wordpress-login` directory to the `/wp-content/plugins/` directory
2. Activate the plugin "Animated Wordpress Login" through the 'Plugins' menu in WordPress
3. Logout and see your newly customized animated login screen
